# Prompt Optimizer · 提示词优化器

> 一键优化你的提示词，让 AI 输出更专业、更准确、更高质量

## 项目简介

Prompt Optimizer 是一个 TRAE Skill，帮助用户将简单、模糊的原始需求，快速转化为结构化、专业化、目标明确的高质量提示词，从而显著提升 AI 输出的质量和准确性。

## 核心特性

- 🚀 **一键优化**：输入 `/optimize` + 原始 prompt，即刻获得专业优化版本
- 🎯 **三大框架**：CRISPE（全面型）、ROSES（创意文案型）、结构化多层优化（通用型）
- 🤖 **智能识别**：自动分析内容，匹配最佳优化框架
- 📊 **对比展示**：优化前后对比 + 亮点说明，一目了然
- ✏️ **灵活调整**：支持用户修改调整、切换框架，直到满意
- 📚 **模板库**：内置 10 个常用场景专业模板（文案 + 创意）
- ⚡ **直接执行**：确认后可直接用优化后的 prompt 执行任务

## 快速开始

```
/optimize 帮我写一个奶茶店的宣传文案
```

### 指定框架

```
/optimize --crispe 写一份产品需求文档
/optimize --roses 写一篇小红书种草笔记
/optimize --structured 帮我整理旅行清单
```
