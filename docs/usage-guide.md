# 使用指南 · Prompt Optimizer

## 快速开始

### 基本用法
在输入框中输入 `/optimize` 加上你想要优化的提示词即可。

```
/optimize 帮我写一篇关于AI的文章
```

### 指定框架
你也可以指定使用特定的优化框架：

```
/optimize --crispe 帮我写一个产品需求文档
/optimize --roses 写一个奶茶店的宣传文案
/optimize --structured 帮我整理一份会议纪要
```

| 框架参数 | 对应框架 | 适用场景 |
|---------|---------|---------|
| `--crispe` | CRISPE 全面型 | 复杂任务、专业领域 |
| `--roses` | ROSES 创意文案型 | 文案写作、创意生成 |
| `--structured` | 结构化多层优化 | 通用任务、简单任务 |

## 三大框架详解

### 🎯 什么时候用 CRISPE？

**适用场景**：
- 任务比较复杂，有多个步骤
- 涉及专业领域知识
- 需要精确控制输出质量
- 希望输出稳定可复现

**例子**：
```
/optimize --crispe 帮我分析这份销售数据并给出改进建议
/optimize --crispe 写一份完整的产品需求文档
```

### 🎨 什么时候用 ROSES？

**适用场景**：
- 写文案、广告语、公众号文章
- 创意生成、故事创作
- 品牌营销相关内容
- 需要考虑受众心理的内容

**例子**：
```
/optimize --roses 写一篇小红书护肤产品种草笔记
/optimize --roses 创作一个关于青春的短篇故事
```

### 📋 什么时候用结构化多层优化？

**适用场景**：
- 通用任务，不确定用哪个框架
- 任务比较简单，目标单一
- 想要灵活调整各个维度

**例子**：
```
/optimize --structured 帮我整理一份旅行清单
/optimize --structured 写一封辞职信
```

## 使用技巧

### 技巧 1：提供越多信息，优化效果越好

虽然优化器可以帮你补充很多信息，但如果你能提供更多背景，效果会更好。

**不太好的输入**：
```
/optimize 写文案
```

**更好的输入**：
```
/optimize 写一个奶茶店的开业宣传文案，目标受众是大学生，价格亲民，主打手作鲜果茶
```

### 技巧 2：可以多次迭代优化

第一次优化后，如果你觉得还不够好，可以继续提出修改意见：

```
/optimize 写一个咖啡店广告语
...（优化结果）...
"不错，但风格太文艺了，能不能更活泼更年轻化一点？"
```

### 技巧 3：试试不同框架

同一个 prompt，用不同框架优化可能会有不同的效果。如果对结果不满意，可以切换框架试试：

```
"换成 CRISPE 框架再优化一下"
```

### 技巧 4：优化后直接执行

确认优化结果后，可以直接让 AI 执行，不用再复制粘贴：

```
"直接用这个优化后的 prompt 开始写吧"
```

## 常见问题

### Q1：为什么有时候优化后的 prompt 比原来长很多？
**A**：这是正常的。优化的核心就是补充角色、约束、格式等信息，让 AI 更准确地理解你的需求。更长的 prompt 往往能带来更好的输出质量。

### Q2：我可以保存优化后的 prompt 吗？
**A**：当然可以！你可以把优化后的 prompt 复制保存下来，以后遇到类似任务直接使用或稍作修改即可。

### Q3：自动识别的框架不对怎么办？
**A**：你可以手动指定框架，在输入时加上 `--crispe`、`--roses` 或 `--structured` 参数即可。

### Q4：模板库可以自定义吗？
**A**：当前版本暂不支持用户自定义模板，但你可以通过修改优化结果来满足个性化需求。未来版本会考虑增加自定义模板功能。

### Q5：支持英文 prompt 优化吗？
**A**：支持。系统会自动检测输入语言，英文输入会用英文优化，保持语言一致性。
