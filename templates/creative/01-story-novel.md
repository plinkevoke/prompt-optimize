# 故事 / 小说创作模板

## 适用场景
短篇故事、微小说、小说片段、剧情构思、人物设定等创意写作

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位才华横溢的 [类型] 小说作家，拥有 10 年以上创作经验，擅长 [风格] 的叙事，作品以 [特点] 著称，能够精准把握读者情绪和故事节奏。

### O - Objective（目标）
创作一个关于 [主题] 的 [短篇故事 / 微小说 / 小说开头 / 剧情片段]，目标是 [打动读者 / 制造悬念 / 引发思考 / 带来愉悦阅读体验]，让读者有强烈的代入感和共鸣。

### S - Scenario & Audience（场景与受众）
- **故事类型**：[都市情感 / 悬疑推理 / 科幻未来 / 古风仙侠 / 奇幻冒险 / 治愈温情 / 恐怖惊悚 / 青春校园]
- **目标读者**：[年龄段] [性别]，喜欢 [阅读偏好]
- **阅读场景**：[通勤碎片化阅读 / 睡前阅读 / 周末深度阅读]
- **读者期待**：[情感共鸣 / 烧脑推理 / 爽感体验 / 治愈温暖 / 脑洞大开]
- **时代背景**：[现代都市 / 古代 / 未来世界 / 架空世界 / 民国时期]

### E - Elements & Constraints（要素与约束）
- **核心主题**：[故事主题 / 想表达的核心思想]
- **主要人物**：
  - 主角：[姓名 + 身份 + 性格特点 + 核心诉求]
  - 配角：[姓名 + 身份 + 与主角关系]
- **关键设定**：[世界观设定 / 特殊能力 / 背景故事]
- **必须包含的情节**：
  - [情节1]
  - [情节2]
- **字数**：[500字微小说 / 2000字短篇 / 5000字以上中篇]
- **结局类型**：[开放式 / 圆满结局 / 悲剧结局 / 反转结局 / 留悬念]
- **风格调性**：[轻松幽默 / 深情虐心 / 紧张刺激 / 治愈温暖 / 暗黑压抑 / 烧脑悬疑]
- **禁忌**：[不能出现的内容 / 不能写的情节]

### S - Style & Structure（风格与结构）
- **叙事视角**：[第一人称 / 第三人称全知 / 第三人称有限]
- **故事结构**：
  1. **开篇**：[场景引入 / 人物出场 / 悬念设置] —— 快速抓住读者
  2. **发展**：[冲突出现 / 情节推进 / 人物关系变化]
  3. **高潮**：[核心冲突爆发 / 真相揭晓 / 关键抉择]
  4. **结局**：[收束故事 / 情感升华 / 余韵留白]
- **节奏控制**：[快节奏 / 慢节奏 / 张弛有度]
- **语言风格**：[华丽优美 / 简洁有力 / 口语化 / 文艺范 / 幽默风趣]
- **输出要求**：完整的故事文本 + 一句话故事梗概 + 创作思路说明

## 使用示例
**输入**：`/optimize 写一个关于时间旅行的短篇科幻故事`
**模板匹配**：自动套用本模板，补充具体设定后生成优化 prompt
