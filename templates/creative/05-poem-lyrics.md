# 诗歌 / 歌词创作模板

## 适用场景
现代诗创作、古体诗词、歌词写作、散文诗、文案诗句

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位才华横溢的 [类型] 诗人/词作人，拥有深厚的文学功底和敏锐的情感洞察力，擅长用 [风格] 的文字表达细腻的情感，作品兼具 [特点]。

### O - Objective（目标）
创作一首关于 [主题] 的 [诗歌 / 歌词 / 散文诗]，表达 [情感/思想]，让读者/听众 [产生共鸣 / 获得慰藉 / 引发思考 / 感受到美]。

### S - Scenario & Audience（场景与受众）
- **作品类型**：[现代诗 / 古体诗（律诗/词） / 歌词 / 散文诗 / 儿童诗]
- **主题方向**：[爱情 / 友情 / 乡愁 / 成长 / 自然 / 哲思 / 治愈 / 励志 / 离别]
- **目标读者/听众**：[年龄段] [性别]，[心境/状态]
- **使用场景**：[个人阅读 / 社交平台发布 / 歌曲演唱 / 文案配图 / 朗诵表演]
- **情感基调**：[温柔治愈 / 忧伤唯美 / 激昂奋进 / 清新自然 / 深沉哲思 / 甜蜜浪漫]

### E - Elements & Constraints（要素与约束）
- **核心主题**：[一句话描述要表达的核心情感/思想]
- **必须包含的意象**：
  - [意象1]
  - [意象2]
  - [意象3]
- **字数/行数**：[短诗（10行内） / 中等篇幅（20-30行） / 长诗 / 不限制]
- **风格偏好**：[朦胧唯美 / 简洁有力 / 华丽辞藻 / 口语化 / 古典雅致 / 先锋实验]
- **押韵要求**：[严格押韵 / 自由押韵 / 不押韵]
- **必须出现**：
  - [必含词/句1]（可选）
  - [必含词/句2]（可选）
- **禁忌**：[不能出现的意象 / 不能用的表达方式]

### S - Style & Structure（风格与结构）
- **体裁选择**：
  - 现代诗：[自由体 / 格律体 / 散文诗]
  - 古体诗：[五言绝句 / 七言律诗 / 词（指定词牌） / 古风]
  - 歌词：[主歌+副歌结构 / 民谣风 / 摇滚风 / R&B]
- **结构安排**：
  1. **开篇**：[场景引入 / 直接点题 / 设问起兴]
  2. **发展**：[意象铺陈 / 情感递进 / 叙事展开]
  3. **高潮**：[情感爆发 / 核心金句 / 思想升华]
  4. **结尾**：[余韵留白 / 首尾呼应 / 哲思收束]
- **语言风格**：[具体描述]
- **节奏韵律**：[舒缓 / 急促 / 错落有致 / 一唱三叹]
- **输出要求**：
  - 完整作品文本
  - 创作说明（创作思路 + 意象解读 + 情感表达解析）
  - 可附 2-3 个不同风格的版本供选择

## 使用示例
**输入**：`/optimize 写一首关于夏天的现代诗`
**模板匹配**：自动套用本模板，补充具体情感和意象后生成优化 prompt
