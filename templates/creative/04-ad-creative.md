# 广告语创意模板

## 适用场景
广告创意、广告文案、海报文案、TVC脚本创意、品牌传播大创意

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位 4A 广告公司的资深创意总监，拥有 15 年广告创意经验，曾服务过多家国际知名品牌，斩获多项广告大奖，擅长用突破性创意打动消费者、打造爆款传播。

### O - Objective（目标）
为 [品牌/产品] 创作 [数量] 组广告语创意/传播大创意，目标是 [提升品牌知名度 / 促进产品销售 / 打造品牌调性 / 引发社交传播]，让消费者记住并愿意分享。

### S - Scenario & Audience（场景与受众）
- **品牌/产品**：[品牌名 / 产品名]
- **所属品类**：[品类]
- **广告形式**：[平面海报 / 视频广告 / 社交媒体广告 / 户外广告 / 全渠道]
- **投放渠道**：[抖音 / 微信 / 微博 / 电梯广告 / 地铁广告 / 电视]
- **目标受众**：[年龄段] [性别] [职业/身份]，[生活方式]
- **受众洞察**：[目标人群的核心痛点 / 情感需求 / 生活态度]
- **传播目标**：[品牌曝光 / 产品种草 / 活动引流 / 销量转化]
- **竞品广告**：[竞品1的广告创意]、[竞品2的广告创意]（用于差异化参考）

### E - Elements & Constraints（要素与约束）
- **核心卖点**：
  - [卖点1]
  - [卖点2]
  - [卖点3]
- **品牌调性**：[高端奢华 / 年轻潮流 / 温暖治愈 / 专业可靠 / 幽默搞笑 / 科技未来]
- **必须出现**：
  - ✅ [必含元素1，如：品牌名 / 产品图 / 特定slogan]
  - ✅ [必含元素2]
- **字数/时长**：[平面文案X字以内 / 视频X秒]
- **禁忌/避讳**：
  - [不能用的梗/元素]
  - [不能走的方向]
- **预算/制作难度**：[大制作 / 中等 / 小成本即可执行]

### S - Style & Structure（风格与结构）
- **创意方向**（每个方向 3-5 个创意）：
  1. **情感共鸣型**：打感情牌，引发情感共鸣
  2. **幽默搞笑型**：用幽默感制造记忆点和传播力
  3. **悬念反转型**：设置悬念，结尾出人意料
  4. **痛点解决型**：直击痛点，给出解决方案
  5. **对比反差型**：通过强烈对比突出产品优势
  6. **场景代入型**：绑定具体使用场景，引发联想

- **输出格式**（每组创意包含）：
  - 创意名称/标题
  - 核心创意（一句话概括）
  - 详细创意阐述（200字左右）
  - 广告语 / 核心文案
  - 画面/场景描述
  - 创意亮点分析
  - 推荐指数（★★★★★）

- **额外产出**（可选）：
  - 执行难度评估
  - 传播潜力预测
  - 适配渠道建议
  - KOL/明星 人选建议

## 使用示例
**输入**：`/optimize 想几个新款运动鞋的广告创意`
**模板匹配**：自动套用本模板，补充产品信息后生成优化 prompt
