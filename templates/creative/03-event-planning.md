# 活动策划方案模板

## 适用场景
营销活动策划、品牌活动、线上活动、线下活动、节日活动、用户增长活动

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位资深活动策划总监，拥有 8 年以上品牌营销活动策划经验，操盘过多个百万级曝光的成功活动，擅长从 0 到 1 策划创意十足且可落地执行的活动方案。

### O - Objective（目标）
策划一场关于 [活动主题] 的 [线上/线下/线上线下联动] 活动，目标是 [提升品牌曝光 / 增加用户增长 / 提高销售额 / 增强用户粘性 / 打造品牌声量]，预期达成 [具体KPI]。

### S - Scenario & Audience（场景与受众）
- **活动类型**：[新品发布 / 节日营销 / 用户增长 / 品牌造势 / 会员活动 / 公益活动]
- **活动主题**：[主题名称]
- **活动时间**：[开始时间] - [结束时间]，持续 [天数]
- **活动地点**：[线上平台 / 线下场地]
- **目标受众**：[年龄段] [性别] [职业/身份]，[地域分布]
- **受众画像**：[兴趣爱好]、[消费习惯]、[活动偏好]
- **预算范围**：[大致预算]（可选）

### E - Elements & Constraints（要素与约束）
- **品牌/产品**：[品牌名 / 产品名]
- **核心卖点**：
  - [卖点1]
  - [卖点2]
- **活动预算**：[高 / 中 / 低 / 具体金额]
- **资源支持**：[可用资源，如：KOL资源 / 媒体资源 / 场地资源 / 技术支持]
- **必须包含**：
  - ✅ [必含要素1]
  - ✅ [必含要素2]
- **风格调性**：[年轻潮流 / 高端大气 / 温馨治愈 / 紧张刺激 / 幽默有趣 / 科技感]
- **禁忌/避讳**：[不能出现的元素 / 不能走的方向]
- **可参考的活动**：[竞品活动 / 过往成功活动]

### S - Style & Structure（风格与结构）
活动方案需包含以下模块：

1. **活动概述**
   - 活动背景与目的
   - 活动主题与 slogan
   - 活动时间与地点
   - 目标受众与预期效果

2. **创意核心**
   - 核心创意点（一句话概括活动亮点）
   - 活动玩法设计（详细规则）
   - 用户参与路径
   - 传播机制（如何裂变/分享）

3. **活动流程**
   - 预热期：[时间] + [具体动作]
   - 爆发期：[时间] + [具体动作]
   - 收尾期：[时间] + [具体动作]

4. **宣传推广**
   - 预热渠道与节奏
   - 爆发期推广计划
   - 物料清单（海报/视频/文案等）

5. **执行与分工**
   - 人员分工
   - 时间节点表
   - 风险预案

6. **预算与KPI**
   - 预算分配
   - 预期KPI与衡量标准
   - ROI 预估

7. **创意亮点总结**
   - 3-5 个核心亮点说明

## 使用示例
**输入**：`/optimize 策划一个618电商促销活动方案`
**模板匹配**：自动套用本模板，补充活动详情后生成优化 prompt
