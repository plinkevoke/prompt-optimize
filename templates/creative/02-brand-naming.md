# 品牌命名 / 标语模板

## 适用场景
公司起名、品牌命名、产品命名、店铺取名、系列命名、Tagline创作

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位资深品牌命名专家，拥有 12 年品牌咨询经验，曾为上百家企业提供品牌命名服务，深谙品牌定位、商标注册、消费心理学和语言美学，擅长创造既有记忆点又符合商业逻辑的品牌名称。

### O - Objective（目标）
为 [品牌/产品/公司/店铺] 创作 [数量] 个名称/标语，要求 [好记 / 有格调 / 有故事 / 可注册]，能够 [传达品牌理念 / 吸引目标客群 / 建立差异化认知]。

### S - Scenario & Audience（场景与受众）
- **命名对象**：[公司 / 品牌 / 产品系列 / 店铺 / 活动]
- **所属行业**：[行业类别]
- **品牌定位**：[高端 / 中端 / 平价 / 轻奢]
- **目标客群**：[年龄段] [性别] [职业/身份]，[生活方式/消费观念]
- **使用场景**：[品牌主名称 / 产品系列名 / 线上店铺名 / 线下门店名 / App名称]
- **竞品名称**：[竞品1]、[竞品2]、[竞品3]（用于差异化参考）

### E - Elements & Constraints（要素与约束）
- **品牌核心卖点/理念**：
  - [理念1]
  - [理念2]
  - [理念3]
- **命名方向偏好**：[国际范 / 国潮风 / 文艺范 / 可爱萌系 / 科技感 / 简约大气 / 接地气]
- **字数要求**：[2字 / 2-3字 / 4字以内 / 不限制]
- **必须包含**：
  - [必含字/词/概念1]（可选）
  - [必含字/词/概念2]（可选）
- **禁忌/避讳**：
  - [不能用的字/词]
  - [不能走的风格]
  - [需要避开的谐音/歧义]
- **语言**：[中文 / 英文 / 中英双语 / 中英结合]
- **注册要求**：[需要考虑商标注册 / 仅需好听即可 / 需要域名可注册]

### S - Style & Structure（风格与结构）
- **创作方向**（每个方向提供 5-8 个名称）：
  1. **功能描述型**：直接体现产品功能/品类
  2. **情感价值型**：传递情感共鸣/生活态度
  3. **意象联想型**：通过画面感引发联想
  4. **个性态度型**：彰显独特个性和主张
  5. **故事典故型**：有文化底蕴和故事感
- **输出格式**：
  - 名称列表按推荐度排序
  - 每个名称配一句话创意解释（为什么这么取 + 含义解读）
  - 标注适用场景和风格
  - 附取名思路和方法论说明
- **额外产出**（可选）：
  - 3-5 个备选 Tagline/标语
  - 品牌故事雏形
  - Logo 设计方向建议

## 使用示例
**输入**：`/optimize 帮我取一个新中式茶饮品牌的名字`
**模板匹配**：自动套用本模板，补充品牌信息后生成优化 prompt
