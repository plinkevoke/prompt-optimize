# 短视频脚本 / 口播文案模板

## 适用场景
抖音/快手短视频脚本、视频号口播文案、Vlog脚本、直播话术

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位资深短视频内容策划师，拥有 5 年以上短视频创作经验，精通抖音/快手/视频号的平台算法和用户喜好，擅长创作高完播率、高互动率的爆款视频脚本。

### O - Objective（目标）
创作一条关于 [主题] 的短视频脚本/口播文案，目标是 [提高完播率 / 增加点赞关注 / 引导转化 / 打造人设]，视频时长约 [时长]。

### S - Scenario & Audience（场景与受众）
- **发布平台**：抖音 / 快手 / 视频号 / B站 / 小红书视频
- **视频时长**：[15秒 / 30秒 / 60秒 / 3分钟以内]
- **视频类型**：[口播讲解 / 剧情演绎 / 好物展示 / 教程教学 / Vlog记录]
- **目标观众**：[年龄段] [性别] [职业/身份]
- **观众画像**：[兴趣标签]、[内容偏好]
- **观看场景**：[碎片化刷视频 / 主动搜索学习 / 睡前娱乐]

### E - Elements & Constraints（要素与约束）
- **核心主题**：[视频主题]
- **核心信息点**：
  - [信息点1]
  - [信息点2]
  - [信息点3]
- **必须包含**：
  - ✅ [必含要素1]
  - ✅ [必含要素2]
- **风格调性**：[轻松幽默 / 专业干货 / 情感共鸣 / 高能反转 / 治愈温暖]
- **人设/口吻**：[专家型 / 闺蜜型 / 搞笑型 / 励志型]
- **禁忌**：[不能出现的内容 / 不能说的话]

### S - Style & Structure（风格与结构）
- **开头钩子（前3秒）**：[痛点提问 / 惊人结论 / 悬念预告 / 反差开场] —— 必须在前3秒抓住注意力
- **正文结构**：
  - 第 1 段：[内容1]（约 X 秒）
  - 第 2 段：[内容2]（约 X 秒）
  - 第 3 段：[内容3]（约 X 秒）
- **结尾引导**：[互动提问 / 关注引导 / 下期预告 / 行动号召]
- **画面/运镜建议**：[每段对应的画面描述]
- **字幕重点**：标出需要加字幕强调的关键词
- **BGM建议**：[音乐风格 / 具体曲目推荐]
- **话题标签**：[5-8个相关话题标签]

## 使用示例
**输入**：`/optimize 写一个30秒减肥产品抖音短视频脚本`
**模板匹配**：自动套用本模板，补充产品信息后生成优化 prompt
