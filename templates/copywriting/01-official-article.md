# 公众号/自媒体文章模板

## 适用场景
公众号推文、知乎专栏、今日头条、博客文章等长图文内容

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位资深新媒体内容主编，拥有 8 年以上公众号/自媒体运营经验，擅长撰写 [领域] 类深度好文，深谙读者心理和平台算法，曾打造多篇 10W+ 爆款文章。

### O - Objective（目标）
创作一篇关于 [主题] 的公众号文章，目标是 [吸引目标读者阅读 / 提升品牌认知 / 引导转化 / 引发共鸣和讨论]，预期阅读量和转发量达到同类文章中上水平。

### S - Scenario & Audience（场景与受众）
- **发布平台**：微信公众号 / 知乎 / 今日头条 / 小红书长文
- **目标读者**：[年龄段] [性别] [职业/身份]，关注 [兴趣领域]
- **阅读场景**：通勤路上 / 午休时间 / 睡前浏览 / 工作间隙
- **读者痛点**：[具体痛点1]、[具体痛点2]、[具体痛点3]

### E - Elements & Constraints（要素与约束）
- **核心主题**：[文章主题]
- **核心观点**：[文章要传递的主要观点]
- **必须包含**：
  - [要素1]
  - [要素2]
  - [要素3]
- **字数要求**：[800-1200字 / 1500-2000字 / 3000字以上]
- **风格调性**：[干货实用型 / 情感共鸣型 / 幽默风趣型 / 专业深度型 / 故事叙述型]
- **关键词**：[SEO关键词1]、[SEO关键词2]、[SEO关键词3]

### S - Style & Structure（风格与结构）
- **文章结构**：
  1. 标题（提供 3-5 个备选，含数字型、悬念型、对比型等）
  2. 开头（场景引入 / 痛点提问 / 故事开篇，吸引读者继续阅读）
  3. 正文（分 3-5 个小节，每节有小标题，层层递进）
  4. 结尾（总结升华 + 金句 + 互动引导）
- **配图建议**：[配图数量和方向]
- **互动引导**：[点赞 / 评论 / 转发 / 关注 / 点击阅读原文]

## 使用示例
**输入**：`/optimize 写一篇关于年轻人存钱的公众号文章`
**模板匹配**：自动套用本模板，补充具体细节后生成优化 prompt
