# 小红书种草文案模板

## 适用场景
小红书笔记、种草笔记、好物分享、产品测评等

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位资深小红书博主，深耕 [领域] 赛道 3 年+，粉丝 [量级]，深谙小红书流量密码和用户喜好，擅长用真实体验感的口吻创作高互动种草笔记。

### O - Objective（目标）
为 [产品/品牌/好物] 创作一篇小红书种草笔记，目标是 [提升产品认知 / 引导购买转化 / 增加品牌曝光]，预期获得高赞藏和评论互动。

### S - Scenario & Audience（场景与受众）
- **发布平台**：小红书
- **目标用户**：[年龄段] 岁 [性别]，[职业/身份]，生活在 [城市层级]
- **用户画像**：关注 [兴趣标签]，追求 [生活态度]，消费能力 [消费水平]
- **使用场景**：[具体使用场景1]、[具体使用场景2]
- **核心痛点**：[用户痛点1]、[用户痛点2]

### E - Elements & Constraints（要素与约束）
- **产品名称**：[产品名]
- **产品品类**：[品类]
- **核心卖点**：
  - [卖点1]
  - [卖点2]
  - [卖点3]
- **价格区间**：[价格]
- **笔记类型**：[好物分享 / 开箱测评 / 教程攻略 / 对比横评 / 避坑指南]
- **必须包含**：
  - ✅ [必含要素1]
  - ✅ [必含要素2]
- **风格调性**：[真诚分享 / 闺蜜种草 / 专业测评 / 搞笑沙雕 / 高级质感]
- **字数**：[300-500字 / 500-800字]

### S - Style & Structure（风格与结构）
- **标题**：提供 5 个爆款标题备选，包含以下类型：
  - 数字型（含具体数字/价格/数量）
  - 痛点型（直击用户痛点）
  - 对比型（前后对比/好坏对比）
  - 悬念型（引发好奇）
  - emoji 型（活泼吸睛）
- **正文结构**：
  1. 开头钩子（痛点共鸣 / 惊人发现 / 真实场景）
  2. 产品介绍（是什么 + 为什么推荐）
  3. 使用体验（真实感受 + 细节描述 + 效果展示）
  4. 优缺点分析（真实客观，增加可信度）
  5. 购买建议 + 总结
- **标签**：提供 8-12 个精准标签（含大流量词 + 精准长尾词）
- **emoji**：适当使用，每段 1-2 个，增加活泼感

## 使用示例
**输入**：`/optimize 写一篇小红书口红种草笔记`
**模板匹配**：自动套用本模板，补充具体产品信息后生成优化 prompt
