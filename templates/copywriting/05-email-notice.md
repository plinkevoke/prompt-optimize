# 邮件 / 通知 / 公告模板

## 适用场景
商务邮件、工作通知、公司公告、活动通知、邀请函等正式书面沟通

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位专业的商务文书撰写专家，拥有多年企业行政/市场/HR 工作经验，精通各类正式文书的写作规范和沟通技巧，能够根据不同场景和对象调整语气和措辞。

### O - Objective（目标）
撰写一封关于 [主题] 的 [邮件/通知/公告]，目标是 [清晰传达信息 / 促使收件人行动 / 维护良好关系 / 达成合作]，确保信息准确无误且易于理解。

### S - Scenario & Audience（场景与受众）
- **文书类型**：[商务邮件 / 内部通知 / 公司公告 / 活动邀请函 / 道歉信 / 感谢信 / 邀约邮件]
- **发件人身份**：[发件人职位 / 部门 / 公司]
- **收件人**：[收件人身份 / 职位 / 与发件人关系]
- **收件人数量**：[一对一 / 小范围 / 全公司 / 公开公告]
- **发送场景**：[日常工作沟通 / 重要事项通知 / 活动邀请 / 危机公关 / 商务合作]
- **收件人可能的反应/顾虑**：[可能的疑问 / 抵触情绪 / 关注点]

### E - Elements & Constraints（要素与约束）
- **核心主题**：[主题]
- **必须包含的信息**：
  1. [信息点1]
  2. [信息点2]
  3. [信息点3]
- **关键时间/地点**：[时间] / [地点]
- **行动要求**：[是否需要收件人行动 / 什么行动 / 截止时间]
- **语气风格**：[正式严肃 / 友好亲切 / 专业客观 / 热情洋溢 / 诚恳致歉]
- **字数限制**：[简明扼要（100字内） / 适中（300字左右） / 详细（500字以上）]
- **禁忌/避讳**：[不能提及的内容 / 不能使用的措辞]

### S - Style & Structure（风格与结构）
- **结构组成**：
  1. **标题/主题**：清晰明确，一眼看懂内容（邮件主题不超过20字）
  2. **称呼**：根据收件人关系选择合适称呼
  3. **开头**：开门见山说明来意 / 礼貌问候
  4. **正文**：
     - 第一段：核心信息（结论先行）
     - 第二段：详细说明 / 背景补充 / 原因解释
     - 第三段：行动要求 / 联系方式 / 注意事项
  5. **结尾**：感谢 / 期待回复 / 祝福语
  6. **署名**：发件人 + 日期
- **排版要求**：[分段清晰 / 要点加粗 / 使用列表 / 正式格式]
- **附件说明**：[是否有附件 / 附件名称]

## 使用示例
**输入**：`/optimize 写一封公司年会活动通知邮件`
**模板匹配**：自动套用本模板，补充活动详情后生成优化 prompt
