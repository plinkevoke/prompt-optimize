# 产品广告语 / 品牌 Slogan 模板

## 适用场景
品牌广告语、产品 slogan、品牌标语、宣传口号、品牌定位语

## 模板结构（ROSES 框架）

### R - Role（角色）
你是一位顶级品牌创意总监，拥有 15 年品牌策划经验，曾为多家知名品牌打造过经典 slogan，深谙品牌定位与消费者心理，擅长用一句话击中人心。

### O - Objective（目标）
为 [品牌/产品] 创作 [数量] 条广告语/slogan，要求 [好记 / 有格调 / 有传播力 / 突出卖点]，能够 [提升品牌记忆度 / 传达品牌理念 / 促进销售转化]。

### S - Scenario & Audience（场景与受众）
- **品牌名称**：[品牌名]
- **品牌/产品品类**：[品类]
- **使用场景**：[广告语使用场景，如：产品包装 / 广告投放 / 社交媒体 / 门店招牌 / 电视广告]
- **目标受众**：[年龄段] [性别] [职业/身份]
- **受众心理**：[目标用户的心理诉求 / 情感需求 / 价值观]
- **竞品 slogan**：[竞品1 slogan]、[竞品2 slogan]（用于差异化参考）

### E - Elements & Constraints（要素与约束）
- **品牌核心卖点**：
  - [卖点1]
  - [卖点2]
  - [卖点3]
- **品牌调性**：[高端奢华 / 年轻活力 / 温暖治愈 / 专业可靠 / 潮流先锋 / 简约自然]
- **字数限制**：[4字 / 6-8字 / 10字以内 / 不限制]
- **必须包含**：
  - [必含词/概念1]（可选）
  - [必含词/概念2]（可选）
- **禁忌/避讳**：[不能出现的词 / 不能走的风格]
- **语言**：[中文 / 英文 / 中英双语]

### S - Style & Structure（风格与结构）
- **创作方向**（提供多个方向，每个方向 3-5 条）：
  1. **功能卖点型**：直接突出产品功能/效果
  2. **情感共鸣型**：打感情牌，引发情感共鸣
  3. **态度主张型**：传递品牌态度和价值观
  4. **场景联想型**：绑定具体使用场景
  5. **对比反差型**：通过对比制造记忆点
- **输出格式**：
  - 每条 slogan 配一句话创意说明
  - 标注适用场景
  - 按推荐程度排序

## 使用示例
**输入**：`/optimize 帮我想一个咖啡品牌的slogan`
**模板匹配**：自动套用本模板，补充品牌信息后生成优化 prompt
